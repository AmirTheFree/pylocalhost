برنامه ای برای سرو کردن و رندر فایل ها با سرور در لوکال هاست
این یک فایل منیجر مبتنی بر وبه که روت پثش ~/Pylocalhost کامپیوتر شماست.
به درد سرو کردن اچ تی ام ال-آژاکس-پی اچ پی و هرچیزی که نیاز به یک وب سرور داره میخوره.

نکته: پای لوکال هاست جیسون ای پی آی شامل اطلاعات پوشته بهون برمیگردونه اگر آخر یو آر ال پوشه عبارت ?api=true رو اضافه کنید.
برای مثال:
http://localhost/folder1/folder2/ (که فایل منیجر گرافیکی رو نشون میده) --> http://localhost/folder1/folder2/?api=true (جیسون برمیگردونه)

استفاده کامند لاین

$ sudo pylh <command>

Commands:
  • start --> شروع به کار پای لوکال هاست
  • stop --> توقف کار پای لوکال هاست
  • restart --> بارگیری مجدد فایل های پیکربندی و شروع مجدد پای لوکال هاست
  • enable --> فعال سازی قابلیت شروع پای لوکال هاست هنگام روشن شدن رایانه شما
  • disable --> غیر فعال سازی قابلیت شروع پای لوکال هاست هنگام روشن شدن رایانه شما
  • help --> نمایش راهنما
  • ? --> نمایش راهنما

راهنمای تنظیمات

Allow requests from unknown hosts:
گزینه الاو رو انتخاب کنید اگر میخواید پای لوکال هاست اجازه بده از هاست های غریبه بهش دسترسی داشته باشن
اگر گزینه دیس الاو رو انتخاب کنید پای لوکال هاست فقط از لوکال هاست کامپیوتر شما یا ای پی لوکال هاست از کامپیوتر شما که همون ۱۲۷.۰.۰.۱ هست در دسترس خواهد بود و افراد دیگه از شبکه به اون دسترسی نخواهند داشت
نکته: این گزینه به طور پیشفرض دیس الاو شده
اخطار: اگر گزینه الاو رو انتخاب کنید پای لوکال هاست شما در شبکه به اشتراک گذاشته خواهد شد پس حواستون باشه دارید چی کار میکنید.
نکته: حتی اگر گزینه الاو رو هم انتخاب کنید بخش تنظیمات برای غریبه ها قابل دسترسی نخواهد بود.

Show hidden files:
شو رو انتخاب کنید اگر میخواید پای لوکال هاست فایل های پنهان یونیکس که با نقطه شروع میشن رو به شما نشون بده
نکته: این گزینه به طور پیشفرض غیرفعال است

Default editor theme:
شما میتوانید تم پیشفرض را برای ویرایشگر پای لوکال هاست که از آن برای ویرایش یا ایجاد فایل ها استفاده میکنید انتخاب کنید

راهنمای نوار ابزار

شما میتونید یک نوار ابزار بالای پوشه ها و فایل ها ببینید که بخش های مختلفی داره:

شما رو به صفحه اصلی و خانه بر میگردونه

شما رو به یک پوشه عقب تر بر میگردونه

این گزینه پوشه ای که الان درش هستید رو با فایل منیجر پیشفرض سیستمتون باز میکنه و اگر موفق نشه هیچ اتفاقی نخواهد افتاد

این گزینه پوشه ای که توش هستید رو به جای نشون دادن محتوای داخلش با سرور سرو میکنه که یا فایل ایندکس رو نشون میده و یا صفحه فوربیدن

یک نوت بوک ژوپیتر روی پوشه فعلی ایجاد میکنه(درصورتی که موقع نصب کردن قبول کرده باشید که ژوپیتر رو هم نصب کنه)اگر هم ناموفق شد هیچ اتفاقی نمی افته

تمام نوت بوک های ژوپیتر فعال رو استاپ میکنه و در صورتی که ناموفق بود هیچ اتفاقی نمی افته و ممکنه هم کلا کار نکنه

راهنمای منوی کلیک راست

با کلیک کردن روی یک فایل،پوشه یا ... شما میتونید لیستی از کار هایی رو که میتونید با اون فایل یا ... انجام بدید مشاهده کنید
نکته: آیتم هایی که * دارند فقط از کامپیوتری که پای لوکال هاست در آن نصب شده قابل استفاده هستند (فقط از ۱۲۷.۰.۰.۱ یا localhost) حتی اگر شما دسترسی از هاست های بیگانه رو فعال کرده باشید

View source >>> سورس فایل رو در قالب متن خام نشون میده
Download >>> دانلود آیتم
Delete * >>> حذف آیتم
Rename * >>> تعویض نام آیتم
Edit * >>> ویرایشگر رو برای ویرایش فایل باز میکنه
Open with OS * >>> آیتم رو با نرم فازار پیشنهادی سیستم عامل شما باز میکنه و اگر موفق نشه هیچ اتفاقی نمیافته
Run script * >>> فایل رو اجرا میکنه (فقط فایل های پایتون یا جاوا اسکریپت) و خروجی رو نشون میده
