Persian | فارسی
NOTE : Pylocalhost will return you a JSON api including info of contents of a directory if you add ?api=ture at the end of directory urls.
For example:
http://localhost/folder1/folder2/ (Returns the GUI file manager) --> http://localhost/folder1/folder2/?api=true (Returns JSON)

CLI Usage:

$ sudo pylh <command>

Commands:
  • start --> Start Pylocalhost
  • stop --> Stop Pylocalhost
  • restart --> Reload config files & restart Pylocalhost
  • enable --> Enable run as start up for Pylocalhost
  • disable --> Disable run as start up for Pylocalhost
  • help --> Show help
  • ? --> Show help

Settings help

Allow requests from unknown hosts:
Allow this option if you want to share your Pylocalhost over network
If you disallow this option Pylocalhost will be available only from http://localhost or http://127.0.0.1 so other people cannot accesss to your pylocalhost and it is only accessible form your computer and localhost(or its ip : 127 ...)
NOTE: This option is disallowed by default
WARNING: If you allow this option your Pylocalhost will be shared over network be sure and aware about what are you doing.
NOTE: Settings part won't be accessible by other people even if you allow this option.

Show hidden files:
Select Show option if you want Pylocalhost to show Unix hidden files (starting with . )
NOTE: This option is disabled by default

Toolbar help

You can see a toolbar above the files and folders that has some sections:
1 - Home icon: Return you back to the home page of localhost which shows files in your ~/Pylocalhost .
2 - Back icon: Keeps you to the parent directory.
3 - Red folder icon: Opens the current directory using your systems default file browser/manager. If it fails nothing will happen :)
4 - Red folder icon in green Nginx icon : Serves current directory using nginx so it will show you the index file or forbidden page.
5 - Jupyter icon : Starts a Jupyter notebook on current directory(If you had accepted Jupyter installation when you was installing PyLocalHost).If it fails nothing will happen.
6 - Delete icon on Jupyter icon: Stop all running Jupyter notebooks.If it fails nothing will happen(It may not work).
7 - Adress bar: Shows the path of current directory.
